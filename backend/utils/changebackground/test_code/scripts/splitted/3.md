# 积分的面积累积图像

## 典型示例

考虑函数 $f(t) = \sin(t)$,计算其从 $0$ 到 $x$ 的面积累积函数:

$$A(x) = \int_0^x \sin(t)dt = -\cos(t)\Big|_0^x = 1 - \cos(x)$$

**图像分析:**
- 在 $[0, \pi]$ 区间,$\sin(t) > 0$,$A(x)$ 从 $0$ 增长至 $2$
- 在 $[\pi, 2\pi]$ 区间,$\sin(t) < 0$,$A(x)$ 从 $2$ 回落至 $0$
- 在 $x = \pi$ 处,$A(x)$ 达到最大值 $2$,对应 $\sin(\pi) = 0$
